### 大指数快速幂
#### 题目描述
给定整数$x,y$，你需要输出$x^y mod p$的值

##### 输入描述
> 第一行整数$t (1 \le t \le 10)$表示接下来有$t$组测试用例，每个测试用例有三行整数$x,y,p$
>
> $0 \le x \le 100000, 0 \le y \le 10^{100000},100000 \le p \le 1000000007$

##### 输出描述
> 输出t行，每行一个整数代表$x^y mod p$的值

**示例1**

输入
```
5
3
4
998244353
2
10
998244353
0
100
998244353
1
100
998244353
4
100
1000000007
```

输出
```
81
1024
0
1
499445072
```
